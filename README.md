# Azure Function (PowerShell) Template

A template repository for building and deploying Azure Functions written in
PowerShell, with local testing and CI/CD to Azure already wired up.

**→ See [TEMPLATE.md](TEMPLATE.md) for everything you need to use this template:**
what's included, what to rename, what GitHub values to set, and how the
deployment pipeline works.
